Menu: (operación interactiva)

Modules >> Visualization >> 2D Viewers >> View2D

1.Conectar la imagen sobre la cual se va a seleccionar el componente conexo a la entrada de este View2D.

2.Dar clic derecho sobre el View2D, abrir el menú View 2D Options y habilitar la opción Show Inventor Inputs.

Modules >> Visualization >> Interaction >> SoView2DMarkerEditor.

1.Conectar la salida self del SoView2DMarkerEditor con la entrada inInvPreLUT del View2D.

2.Dar doble clic en el módulo y seleccionar en DrawingColor un color diferente a blanco. Esto permitirá ver el punto seleccionado dentro de un objeto.

Modules >> Segmentation >> Region >> ConnectedComponents

1.Conectar la imagen sobre la cual se va a seleccionar el componente conexo a la entrada de este módulo.

2.Conectar la salida outXMarkerList del SoView2DMarkerEditor (cuadrada) a la segunda entrada (cuadrada) de ConnectedComponents.

3.Conectar la salida output1 de este módulo ConnectedComponents a un View2D (diferente al utilizado en el primer paso).

4.Dar doble clic en el módulo ConnectedComponents presionar los botones Update Analysis y Update Component Selection para ver el resultado.

5.Hacer click sobre el o los objetos binarios blancos que desea seleccionar en el primer View2D. Cada vez que seleccione un nuevo punto, repita el paso anterior (click en los botones Update Analysis y Update Component Selection del ConnectedComponents).
