Paleta de color

La paleta de color, o LUT (Look Up Table), es una tabla que otorga una equivalencia entre el valor numérico de un pixel, y el color de visualización de este punto.

El rol de una paleta de color es:

- realzar ciertas informaciones

- mejorar la imagen

- definir pseudo-colores para facilitar la interpretación de la imagen

- efectuar interactivamente ciertas transformaciones del histograma, umbralización...
