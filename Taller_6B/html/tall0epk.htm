Programación en MeVisLab

Imágenes de trabajo : rondelle.png ; spot.png

Ejercicio:

1- Vaya al Menú File y seleccione la opción Run Project Wizard

2- Antes de empezar se debe crear el paquete donde se guardará el código. Haga click en New Package.

·Dé un nombre al paquete en Package Group.

·Indique el directorio en el que se guardará en Target Directory.

·Create

3- Vuelva al Project Wizard menu y seleccione la opción ML Module. Esta opción lo guiará por los pasos necesarios para construir un módulo MeVisLab.

4- En la ventana siguiente defina: nombre del módulo, nombre del proyecto y el páquete (el que acaba de seleccionar). Dar Next.

5-La siguiente ventana permite definir el número de imágenes de entrada y salida que tendrá el módulo. Definir dos imágenes de entrada, y una de salida.

6- La última ventana permite definir parámetros diferentes a imágenes. En este caso se va a añadir un valor numérico constante que se sumará a las imágenes. Para esto se debe dar clic en New.

·Dar el nombre al parámetro y su tipo.

·Los demás campos son opcionales..

7- Si desea guardar la configuración, dar click en Save Settings.

8- Dar click en Create para que se genere el código.

9- Se creará un archive .vcproj. Ábralo y modifique el código fuente para que se puedan sumar píxel a píxel las dos imágenes y a estas se les adicione la constante.

10- Compile en modo Release

12- Reinicie MeVisLab. Busque el módulo que acaba de crear utilizando la herramienta de búsqueda.

13- Cree una red que permita probar la caja. Utilice las imágenes de trabajo.

14- Guarde la red y cierre.
