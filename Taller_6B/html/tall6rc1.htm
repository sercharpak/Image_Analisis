Imagen de distancia

Imagen de trabajo: objects.png

Une imagen de distancia es calculada a partir de una imagen binaria.

Esta operación asigna a cada pixel de la imagen, la distancia euclidiana de éste al objeto (blanco) más cercano.

El cálculo exacto es difícil y largo de realizar, pero existen algoritmos suficientemente rápidos que permiten encontrar buenas aproximaciones de la distancia euclidiana.

Ejercicio:

1 Cree una nueva red en MeVisLab.

2- Añada a esta red un módulo ImageLoad. Abra la imagen objects.png.

3- Esta imagen objects.png es una imagen RGB, conviértala a niveles de gris utilizando un módulo ColorModelConverter.

4- Umbralice esta imagen para separar los objetos del fondo (umbral=70).

5- Calcule la imagen de distancia. Puede efectuar una expansión del contraste sobre la imagen de distancia para visualizar mejor el resultado. Ajuste el Window/Level

6- Para localizar mejor los objetos en la imagen de distancia, calcule el máximo entre esta imagen (sin la expansión del contraste) y la imagen binaria obtenida en 3. Utilizando la imagen resultante, determine la distancia mínima entre los dos lápices (utilice los valores de nivel de gris de los pixeles desplegados en la parte superior de la ventana del programa, cuando se desplaza el cursor sobre la imagen).

7- Guarde la red y cierre.
