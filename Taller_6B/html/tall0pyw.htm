Umbralización doble

Imágenes de trabajo: rondelle. png, angio.png

Ejercicio:

1- Cree una nueva red en MeVisLab.

2- Añada a esta red dos módulos ImageLoad. Abra las imágenes rondelle.png y angio.png. Visualícelas.

3- Añada dos módulos de umbralización IntervalThresh. Conecte la salida de cada ImageLoad a la entrada de cada IntervalThresh. Añada dos nuevos módulos de visualización y conéctelos a las salidas de cada módulo IntervalThresh.

4- Se desea aplicar una umbralización doble. La imagen resultante debe contener tres clases, cuyos niveles de gris son 0, 128 y 255 respectivamente. Para esto se debe hacer uso de los siguientes campos del módulo IntervalThresh:

·Threshold Interval - Min: Permite definir el umbral inferior.

·Threshold Interval - Max: Permite definir el umbral superior.

·Fill values - Below: Especifica el valor de los voxeles de salida que se encuentren por debajo del umbral inferior.

·Fill values - Inner: Especifica el valor de los voxeles de salida que se encuentren dentro del rango [min max] del umbral.

·Fill values - Above: Especifica el valor de los voxeles de salida que se encuentren por encima del umbral superior.

·Nota: Los límites del umbral [min max] también pueden ser controlados a través de los parámetros center y width. Específicamente, Min = center - width / 2 y Max = center + width / 2.

5- Evalúe sobre las imágenes de trabajo rondelle.png y angio.png, utilizando como umbrales bajo y alto los valores 70 y 140 respectivamente. Visualice el resultado.

6- Deduzca de estos ejemplos el rol y las condiciones de utilización de la umbralización doble. Cómo es el comportamiento (ventajas y desventajas) con respecto a: conectividad, extracción de objetos de formas complejas o ramificadas y calidad de los contornos o fronteras de los objetos extraídos.
